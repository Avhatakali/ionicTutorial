HOW TO START WITH IONIC STEP BY STEP

  • 1 st download latest Node.js, install it

    • check if yuh have node.js installed on your terminal, run the command $ node -v, and $ npm -v, To confirm that you have npm installed
    Note: update your npm before yuh can create your first ionic app $npm install npm@latest -g

  • 2 nd Open your command prompt to install Cordova and Ionic:

    • $npm install -g cordova ionic

  • 3rd create app with the following command depending on what yuh plan to create
    • this command create a blank app

      • ionic start myApp blank

    • this command create Tabs app

      • ionic start myApp tabs

    • this command create sidemenu app

      • ionic start myApp sidemenu

  • 4 thNavigate APP folder, Then type the following code to serve your app on the web browser

    • $ ionic serve


  • celebrate, yuh just created your first ionic app
